# Workbook do Aluno - Checklists, Diário e Replay

> Use este workbook durante as aulas, no replay e depois de cada sessão.

## 1. Rotina Pre-Market

```text
Data:
Ativo:

[ ] Agenda macro revisada.
[ ] ES e NQ alinhados ou divergentes.
[ ] Value Area anterior marcada.
[ ] POC marcado.
[ ] VAH marcada.
[ ] VAL marcada.
[ ] HVNs principais marcadas.
[ ] LVNs principais marcadas.
[ ] IBH/IBL definidos quando a IB formar.
[ ] AVWAPs relevantes marcadas.
[ ] Gamma revisado, se houver dados.
[ ] Call Wall / Put Wall / Zero Gamma marcados, se houver dados.
[ ] Cenário 1: aceitação.
[ ] Cenário 2: rejeição.
[ ] Operacionais preferenciais do dia.
[ ] Regiões onde NÃO vou operar.
```

## 4. Biblioteca de Prints

Meta mínima:

- 30 prints de Edge HVN.
- 30 prints de Abs/FA.
- 30 prints de Breakout.
- 30 prints de Reteste/Pullback.
- 20 prints de Repair/Rebalance.
- 20 prints de sem trade.

Para cada print:

```text
Ativo:
Data:
Tipo de leitura:
Zona:

O que invalidaria:
Alvo lógico:
Eu operaria? Por quê?
Lição:
```

## 5. Replay

Regras:

- Operar em simulação.
- Escolher uma família por semana.
- Máximo 3 trades por sessão.
- Antes de clicar, explicar em voz alta.
- Se não conseguir explicar, não clicar.
- Depois do trade, salvar print e preencher o diário.

## 6. Critérios Para Avançar

O aluno só deve avançar complexidade quando cumprir:

- 30 prints por família core.
- 20 exemplos explicados em voz alta.
- 10 sessões simuladas sem quebrar regra.
- Diário completo.
- Capacidade de recusar trade ruim.
- Clareza de invalidação e alvo antes da entrada.
